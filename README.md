# knowledge

A personal, navigable knowledge base built from analytical reading —
propositions, arguments, key terms, tensions, open questions, and the
disputes between them, extracted from what I read (in the style of Adler's
*How to Read a Book*) and cross-linked into concepts, argument threads, and
two-sided issues.

**This is a generated static site — there is no build step in this repo.** It is
produced by a deterministic `build-knowledge` generator that lives in a separate
meta-repo and reads a private extraction corpus, then writes this site here.
Served via GitHub Pages at <https://akafred.github.io/knowledge/>.

Open `index.html` for the overview. Sections: Concepts · Tensions · Issues ·
Claims · Threads · Open gaps · Timeline · interactive Graph. All links are
relative, so the site also works opened directly from disk.
