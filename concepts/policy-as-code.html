Policy-as-Code

Governance and authorization rules expressed as executable, version-controlled, unit-tested code evaluated at runtime per request — the dynamic policy layer above static IAM.

Also known as: policy-as-code, policy as code, OPA, Open Policy Agent, Rego, AWS Cedar, Amazon Verified Permissions, HashiCorp Sentinel, Kyverno, OPA Gatekeeper, admission controller, policy engine
