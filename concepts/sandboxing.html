Sandboxing

Isolated execution environments that let untrusted or agent-generated code run with controlled access to filesystem, network and compute.

Also known as: sandbox, sandboxing, sandboxed Python execution, self-hosted sandboxes, GKE Agent Sandbox, gVisor, V8 isolate / Dynamic Worker isolate, unelevated sandbox, elevated sandbox, Zero Trust Architecture (applied to AI tools), agentic workload, WASM sandbox, WebAssembly sandbox
