MCP (Model Context Protocol)

The open protocol standardizing how AI agents connect to external tools, data sources and services, plus the server/gateway ecosystem growing around it.

Also known as: MCP, mcp (model context protocol), MCP (Model Context Protocol), Model Context Protocol, Model Context Protocol (MCP) server, MCP server, MCP servers, MCP server integration, custom MCP server, MCP Gateway, MCP tunnels, MCP export, MCP server (for Sail), MCP Proxy for AWS, Colab MCP Server, shadow APIs / unmanaged MCP servers
