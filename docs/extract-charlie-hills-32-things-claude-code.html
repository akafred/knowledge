Key terms

  • CLAUDE.md Agent Context Files (CLAUDE.md / AGENTS.md)
    A project-level configuration file that sets persistent rules and context for every Claude Code session
  • /commands
    User-defined reusable shortcuts within Claude Code (slash commands)
  • /skills Agent Skills
    Specialist agent configurations within Claude Code, each scoped to a particular job
  • scope limit Guardrails
    Restricting Claude Code's file access to only explicitly named files to prevent unwanted changes
  • /compact Context Rot & Context Limits
    A Claude Code command that summarizes and compresses the conversation context to free up token budget
  • subagents Multi-Agent Orchestration
    Parallel Claude Code agent instances that can handle separate tasks concurrently
